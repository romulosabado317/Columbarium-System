A Web-Based Columbarium Information System
with Interactive Mapping and Digital Records Management
of Maria Della Strada Church



    Overview


This system is designed to manage columbarium records efficiently for Maria Della Strada Church. It provides an organized and digital approach to handling memorial records, along with an interactive mapping feature that helps users easily locate niches.


    Features
interactive mapping of columbarium niches
digital records management
search and retrieval of client information
user-friendly interface
centralized data storage

    System Requirements
Node.js
npm (comes with Node.js)


    Installation Guide

install dependencies
  npm install

  configure environment variables
    create a .env.local file and set up your database and system configuration

  run the system
    npm run dev
