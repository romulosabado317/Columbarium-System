A Web-Based Columbarium Information System with Interactive Mapping and Digital Records Management of Maria Della Strada Church
This project contains everything needed to run and manage the web-based Columbarium Information System locally.
System Description
The system is designed to manage columbarium records efficiently for Maria Della Strada Church. It features interactive mapping for locating niches and digital records management for storing and retrieving information about clients and memorial spaces.
Run Locally
Prerequisites: Node.js
Install dependencies:npm install
Configure environment variables:Set up your database and system configuration in .env.local
Run the system:npm run dev
